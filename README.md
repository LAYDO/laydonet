## First Steps

Upon cloning repo, run through these commands to start development:

```
$ cd laydonet/
$ python -m venv ./venv
$ source venv/Scripts/activate
$ python -m pip install --upgrade pip
$ pip install -r requirements.txt
$ python manage.py runserver
```
